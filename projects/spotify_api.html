Spotify-API-Visualizer

Software zur Visualisierung von Spotify Songdaten in einer 3D Umgebung

Zusammen mit Alexander Reiprich, David Niemann und Daniel Meisler habe ich einen Spotify Visualizer in Python programmiert.
Das Projekt wurde als Endabgabe der Veranstaltung "Datenverarbeitung in der Medienproduktion" im Rahmen meines Studiums durchgeführt. Ziel war es, über ein Python Script auf die Spotify API zuzugreifen und Daten dieser auszulesen und in geeigneter Form in Blender darzustellen. Der Code des Scripts, soll dabei als Blender Addon verfügbar sein. Dargestellt wird eine Großstadt mit Hochhäusern, an welchen Leuchtbildtafeln angebracht sind. Auf diesen werden Statistiken sowie das Coverbild des aktuell abgespielten Songs angezeigt. Zusätzlich zeigt eine Animation der Tageszeit den Fortschritt im Lied, und ein durchfahrender Zug den Namen des momentanen Songs an.
