Portfolio Website

Eine auf HTML und CSS basierende Website

Diese Website wurde im Mai 2022 erstellt und seitdem immer mal wieder weiterentwickelt.
Die Grundversion der Website basiert auf einem HTML Template, welche durch eigenen HTML, CSS und TypeScript Code weiterentwickelt wurde. Ursprünglich war die Entwicklung der Website als Spaß- und Übungsprojekt gedacht. Da sich das Ergebnis sehen lassen kann, nutze ich sie nun als Portfolio Website.
