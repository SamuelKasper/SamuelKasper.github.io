Smart Irrigation System

Ein auf LoRaWAN basierendes intelligentes Bewässerungssystem

Im Rahmen meiner Bachelorarbeit sollte ein intelligentes Bewässerungssystem mithilfe der LPWAN-Technologie LoRaWAN umgesetzt werden. In der selbst entwickelten, responsiven Webanwendung sollen dabei die von den Sensoren gesendeten Daten angezeigt werden. Zudem soll das System anhand der Daten den Wasserbedarf an verschiedenen Stellen ermitteln und wenn nötig die Bewässerung starten. Im Normalfall wird dabei auf das Wasser einer Regenwasserzisterne zugegriffen. Falls jedoch kein Wasser mehr vorhanden ist, erkennt das System den Mangel und schaltet automatisch auf die Grundwasserversorgung um. Um dem System intelligente Eigenschaften zu geben, wurden zusätzlich eine Zeitsteuerung, manuell festlegbare Bodenfeuchtigkeitsgrenzwerte sowie eine Wettersteuerung mittels der OpenWeather API umgesetzt. Diese helfen dabei, den Wasserverbrauch des Systems zu minimieren. Meine Aufgaben waren sowohl die Programmierung der Webanwendung als auch die Umsetzung und Programmierung der Hardwaresteuerung.

Zur Umsetzung wurde The Thigns Network als LoRaWAN Netzwerkserver, sowie Dragino und Milesight Sensoren verwendet. Die Hardware Implementation wurde mithilfe eines Seeeduino LoRaWAN umgesetzt.
