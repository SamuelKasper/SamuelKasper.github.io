Spotify API Visualizer

Zusammen mit Alexander Reiprich, David Niemann und Daniel Meisler habe ich einen Spotify Visualizer in Python programmiert.
Das Projekt wurde als Endabgabe der Veranstaltung "Datenverarbeitung in der Medienproduktion" im fünften Semester (Sommersemester 2022) durchgeführt. Ziel war es, über ein Python Script auf die Spotify API zuzugreifen und Daten dieser auszulesen und in geeigneter Form in Blender darzustellen. Der Code des Scripts, soll dabei als Blender Addon verfügbar sein.
Dargestellt wird eine Großstadt mit Hochhäusern, an welchen Leuchtbildtafeln angebracht sind. Auf diesen werden Statistiken sowie das Coverbild des aktuell abgespielten Songs angezeigt. Zusätzlich zeigt eine Animation der Tageszeit den Fortschritt im Lied, und ein durchfahrender Zug den Namen des momentanen Songs an.


Breaking the Silence

Eine Visual Novel, welche ich im Rahmen der Veranstaltung "Visual Novel" im vierten Studiensemester (Sommersemester 2021) programmiert habe.
Das Projekt wurde mit der hochschuleigenen Engine FudgeStory umgesetzt. Die Veranstaltung wurde während meines Belegungszeitraums zum ersten Mal durchgeführt.
Aufgebaut ist das Spiel wie eine typische Visual Novel mit Entscheidungsbaum und verschiedenen möglichen Enden. In der Geschichte selbst, geht es um Nick, welcher zum Zeitpunkt des Spiels eine schwere Zeit hat, und Mira, den spielbaren Charakter, die versucht Nick dabei zu helfen, diese schwere Zeit zu überstehen.
Das Besondere an diesem Projekt ist, dass wirklich alles, also nicht nur die Story, sondern auch die Bilder, Charaktere und Musik selbstgemacht sind.


Dodge

Ein kleines Geschicklichkeitsspiel, welches ich im vierten Studiensemester (Sommersemester 2021) im Rahmen der Veranstaltung "Prototyping interaktiver Anwendungen" programmiert habe.
Ziel des Spiels ist es, so viele Münzen wie möglich zu sammeln und dabei nicht von den sich bewegenden oder auftauchenden Stacheln treffen zu lassen.
