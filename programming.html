Smart irrigation system | 2023

Im Rahmen meiner Bachelorarbeit sollte ein intelligentes Bewässerungssystem mithilfe der LPWAN-Technologie LoRaWAN umgesetzt werden. In der selbst entwickelten, responsiven Webanwendung sollen dabei die von den Sensoren gesendeten Daten angezeigt werden. Zudem soll das System anhand der Daten den Wasserbedarf an verschiedenen Stellen ermitteln und wenn nötig die Bewässerung starten. Im Normalfall wird dabei auf das Wasser einer Regenwasserzisterne zugegriffen. Falls jedoch kein Wasser mehr vorhanden ist, erkennt das System den Mangel und schaltet automatisch auf die Grundwasserversorgung um. Um dem System intelligente Eigenschaften zu geben, wurden zusätzlich eine Zeitsteuerung, manuell festlegbare Bodenfeuchtigkeitsgrenzwerte sowie eine Wettersteuerung mittels der OpenWeather API umgesetzt. Diese helfen dabei, den Wasserverbrauch des Systems zu minimieren. Meine Aufgabe waren sowohl die Programmierung der Webanwendung als auch die Umsetzung und Programmierung der Hardwaresteuerung.

Sudoku | 2023

Ein dynamisch generiertes Sudoku-Puzzle. Der Algorithmus zum erstellen und lösen des Sudokus basiert auf Backtracking.
Um eine Zahl in ein Feld einzutragen muss diese zuvor in der Zahlenleiste ausgewählt werden. Ist die eingetragene Zahl falsch, wird sie rot markiert.
Der Lösen-Button löst das Sudoku und beendet somit gleichzeitig das Spiel.

Spotify API Visualizer | 2022

Zusammen mit Alexander Reiprich, David Niemann und Daniel Meisler habe ich einen Spotify Visualizer in Python programmiert.
Das Projekt wurde als Endabgabe der Veranstaltung "Datenverarbeitung in der Medienproduktion" im Rahmen meines Studiums durchgeführt. Ziel war es, über ein Python Script auf die Spotify API zuzugreifen und Daten dieser auszulesen und in geeigneter Form in Blender darzustellen. Der Code des Scripts, soll dabei als Blender Addon verfügbar sein.
Dargestellt wird eine Großstadt mit Hochhäusern, an welchen Leuchtbildtafeln angebracht sind. Auf diesen werden Statistiken sowie das Coverbild des aktuell abgespielten Songs angezeigt. Zusätzlich zeigt eine Animation der Tageszeit den Fortschritt im Lied, und ein durchfahrender Zug den Namen des momentanen Songs an.

Breaking the Silence | 2021

Eine Visual Novel, welche ich im Rahmen der Veranstaltung "Visual Novel" im vierten Studiensemester programmiert habe.
Das Projekt wurde mit der hochschuleigenen Engine FudgeStory umgesetzt. Die Veranstaltung wurde während meines Belegungszeitraums zum ersten Mal durchgeführt.
Aufgebaut ist das Spiel wie eine typische Visual Novel mit Entscheidungsbaum und verschiedenen möglichen Enden. In der Geschichte selbst, geht es um Nick, welcher zum Zeitpunkt des Spiels eine schwere Zeit hat, und Mira, den spielbaren Charakter, die versucht Nick dabei zu helfen, diese schwere Zeit zu überstehen.
Das Besondere an diesem Projekt ist, dass wirklich alles, also nicht nur die Story, sondern auch die Bilder, Charaktere und Musik selbstgemacht sind.
